归档及文件压缩

df

df -i

查看inode号

tar

tar cvf dir1 dir1.tar

-c 创建一个新存档

-x 从现有存档提取

-t 列出存档目录

-v 详细信息

-f 指定创建的文件名

-p 保留文件和目录的权限，而不去除umask

同名文件自动覆盖，不会询问

tar -tf dir1.tar

列出压缩包内容

tar rf /etc/passwd dir1.tar

添加文件到已有压缩包

tar xf dir1.tar

解压压缩包

tar xf dir1.tar -C /tmp

指定目录解压

三种压缩方式

gzip –选项（z）tar czvf dir1 dir1.tar.gz

bzip2 –选项（j）tar cjvf dir1 dir1.tar.bz2

xz –选项（J）tar cJvf dir1 dir1.tar.xz

rsync

rsync -av /etc /home

YUM

yum clean all; yum makecache; yum repolist all

service

nmcli
