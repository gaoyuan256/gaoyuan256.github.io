骚年-Markdown/RHEL-7/Minimal安装图形化界面

挂载镜像源

配置YUM

yum -y groupinstall “Server with GUI”

即可安装。

安装完成后，设置以后直接从图形界面启动：

systemctl set-default graphical.target

systemctl start graphical.target

yum erase gnome-initial-setup -y
