骚年-Markdown/RHEL-7/同步时间ntpdate

同步时间ntpdate

手动执行同步时间

which ntpdate

which hwclock

同步时间

/usr/sbin/ntpdate 192.168.8.8

同步硬件BIOS时间

/usr/sbin/hwclock -w

添加计划任务

crontab -e

0 0 * * * /usr/sbin/ntpdate 192.168.8.8; /usr/sbin/hwclock -w

查看计划任务

crontab -l
